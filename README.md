## How to Use

1. Make sure you already installed and activated **WooCommerce** and **Advanced Custom Fields PRO**.
2. Download the plugin ZIP from GitHub.
3. Go to **WordPress Admin → Plugins → Add New → Upload Plugin**.
4. Upload the ZIP file and activate the plugin.
5. In the admin menu, go to the dedicated **Why Us** settings page.
6. Add your items (image, title, description, link).
7. Save the page.

The items will show up on the **product page**, below the product description.
